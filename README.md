# PO Classification Prediction

This project uses Machine Learning to classify Purchase Orders (PO) based on structured data.  
The system processes input data, performs preprocessing, and predicts the appropriate category.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Features
- Data preprocessing and cleaning
- Feature extraction
- Machine learning model training
- Purchase order classification prediction

## Project Files
- app.py – Main application file
- classifier.py – ML classification logic
- taxonomy.py – Category structure
- prompts.py – Prompt configuration
- requirements.txt – Required libraries

## Installation

Install required libraries:

pip install -r requirements.txt

## Run the Project

python app.py
